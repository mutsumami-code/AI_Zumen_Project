# AI_Zumen_Project

## AIで板金加工の未来をつくる。

板金加工業向けAI図面解析・DXF自動生成システム

---

# プロジェクト理念

AI_Zumen_Project は、板金加工の熟練技術をAIに継承し、
誰でも高品質な加工データを作成できる環境を目指すプロジェクトです。

図面をDXFへ変換するだけではなく、

- 加工ノウハウの継承
- 若手育成
- 生産性向上
- 中小製造業のDX推進

を実現することを目的としています。

---

# プロジェクト概要

AI_Zumen_Project は、板金加工業で使用されるPDF図面をAIが解析し、
加工に必要な情報を理解してDXFデータを生成することを目的としたオープンソースプロジェクトです。

最終目標は、PDF図面からNC加工データ作成までを支援するAIシステムの開発です。

---

# プロジェクトの目的

現在、多くの板金加工会社では、

- PDF図面を確認
- CADで作図
- CAMで加工データ作成
- NCプログラム生成

という工程を人が行っています。

本プロジェクトではAIを活用し、

- 図面解析
- DXF生成
- 曲げ展開
- 加工情報の認識
- CAMデータ作成支援

を実現することを目標としています。
